02113 Tile Generator

Welcome to the 02113 Tile Generator! Here you can quickly design the 32-by-32 pixels tiles to be used as background or sprite.

Instructions: Change the color of the pixels as desired. You can select muliple pixels by keeping CTRL pressed or select a region by keeping SHFT pressed. You can copy the generated binary data into the memory initialization file, or direcly download the .mem file. You can also upload an existing .mem file.
